# TechBuddy

A friendly tech support chatbot for non-tech users.

## Setup

```bash
npm run install:all
cp server/.env.example server/.env   # then add your ANTHROPIC_API_KEY
cp client/.env.example client/.env
npm run dev
```

- Server: http://localhost:3001
- Client: http://localhost:5173

## Modes

- **Chat** — friendly conversational help
- **Show Me** — step-by-step instructions
- **Auto Diagnose** — runs a hardware scan and explains findings
